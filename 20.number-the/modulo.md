# Modular Arithmetic

Modular Arithmetic is basically the operations that deals with the remainders when you divide x by n.
It is represented as: `x mod n`
ex: 75 mod 17= 7

### 1. Modular Arithmetic ‚ÄĒ Rules

Numbers limited to {0, 1, ‚Ä¶, m‚ąí1}. Every x replaced by x mod m.

| | Formula | Notes |
|---|---|---|
| addition | `(x + y) mod m = ((x mod m) + (y mod m)) mod m` | |
| subtraction | `(x ‚ąí y) mod m = ((x mod m) ‚ąí (y mod m) + m) mod m` | +m to avoid negatives |
| multiplication | `(x ¬∑ y) mod m = ((x mod m) ¬∑ (y mod m)) mod m` | |
| exponentiation | `x‚ĀŅ mod m = (x mod m)‚ĀŅ mod m` | |

Congruence: a ‚Č° b (mod n) ‚üļ n | (a ‚ąí b)

Equivalence class: [a]‚āô = { a + kn : k ‚ąą ‚Ą§ }

Formal operations on ‚Ą§‚āô:
- `[a]‚āô +‚āô [b]‚āô = [a + b]‚āô`
- `[a]‚āô ¬∑‚āô [b]‚āô = [ab]‚āô`

### 5. Fast Exponentiation ‚ÄĒ O(log n)

```
x‚Āį       = 1
x‚ĀŅ (even) = (x^(n/2))¬≤ mod m     ‚Üź compute x^(n/2) once, then square
x‚ĀŅ (odd)  = x ¬∑ x^(n‚ąí1) mod m
```

Critical: x^(n/2) computed **once**. Guarantees O(log n).

```cpp
int modpow(int x, int n, int m) {
    if (n == 0) return 1%m;
    long long u = modpow(x, n/2, m);
    u = (u*u)%m;
    if (n%2 == 1) u = (u*x)%m;
    return u;
}
```

### 6. Fermat's & Euler's Theorems

### 7. Modular Inverse

$x^{-1}$ mod m s.t. $x * x^{-1}$ ‚Č° 1 (mod m). Exists iff gcd(x, m) = 1.

| | | |
|---|---|---|
| general m | $x^{-1} = x^{ŌÜ(m)‚ąí1} mod m$ | via Euler |
| m prime | $x‚ĀĽ¬Ļ = x^{m‚ąí2} mod m$ | ŌÜ(m) = m‚ąí1 |
| any m | Extended-Euclid(x, m) | works for composite m too |

Example: 6‚ĀĽ¬Ļ mod 17 = 6^15 mod 17 = 3 ‚Üí check: 6√ó3 = 18 ‚Č° 1 (mod 17) ‚úď

Division: `a/b mod m = a ¬∑ b‚ĀĽ¬Ļ mod m`

Example: 36/6 mod 17 = 2 ¬∑ 3 mod 17 = 6

Inverse doesn't always exist ‚ÄĒ e.g. x=2, m=4: no solution since all multiples of 2 are even.

**CLRS derivation:** Extended-Euclid(a, n) ‚Üí (1, x, y) s.t. ax + ny = 1 ‚üĻ ax ‚Č° 1 (mod n)

Example: a=5, n=11 ‚Üí (1, ‚ąí2, 1) ‚Üí ‚ąí2 ‚Č° 9 (mod 11) is inverse of 5 ‚Üí 5√ó9=45‚Č°1 


$x*x^{-1} mod m = 1$

calculate modular inverse:
$x^{-1}= x^{psi{m}-1}$

if m is prime then:
$x^{-1}=x^{m-2}$

### 9. Modular Exponentiation

oftenly need to efficiently calculate value of $x^{n}  mod m$. This can be done in $O(log n)$ time:

$x^{n}$=
3 cases:

1                   n=0
$x^{n/2} * x^{n/2}$   n is even
$x^{n-1} * x$         n is odd

in case of even n, only once need to calculate value of $x^{n/2}$
Time: O(log n)

$x^n$ mod m:
```cpp
int modpow(int x,int n,int m){
    if(n==0)return 1%m;
    long long u=modpow(x,n/2,m);
    u=(u*u)%m;
    if(n%2==1) u=(u*x)%m;
    return u;
}
```
